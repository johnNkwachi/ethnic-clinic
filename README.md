# Ethnic Integrative Medical Clinic Ltd

Website for **Ethnic Integrative Medical Clinic Ltd** — Integrative, Natural, Rehabilitation Care.

## Live site

Plain HTML, CSS, and JavaScript. No build step required.

## Run locally

- **Option 1:** Open `index.html` in your browser.
- **Option 2:** Serve the folder with a local server:
  ```bash
  npx serve . -p 3000
  ```
  Then open http://localhost:3000

## Pages

- **Home** — `index.html`
- **About** — `about.html`
- **Services** — `services.html`
- **Our Team** — `team.html`
- **Book Appointment** — `appointments.html`
- **Privacy Policy** — `privacy.html`

## Tech

- HTML5, CSS3, vanilla JavaScript
- Mobile-responsive with a hamburger menu on small screens
- Plus Jakarta Sans font (Google Fonts)
